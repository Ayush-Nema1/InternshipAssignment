Frontend Developer Intern Assignment

This repository contains my submission for the Frontend Developer Intern assignment.
The goal was to build a simple but complete full-stack application with authentication and a dashboard, focusing on clean structure, usability, and responsiveness.

Tech Stack

Frontend: React (Vite) + Material UI
Backend: Node.js + Express
Database: MongoDB (running locally)
Authentication: JWT
Styling: Custom CSS with Material UI components

Features

User signup and login
JWT-based authentication
Protected dashboard after login
Create and delete todos
Each user can see only their own todos
Responsive layout for mobile and desktop
Basic server-side validation
Passwords stored securely using hashing 

Project Structure

/frontend – React application (Vite)
/backend – Node.js + Express API

How to Run the Project Locally
Backend
cd backend
npm install
node server.js

Frontend
cd frontend
npm install
npm run dev
